# 📈 Building a Cyber Security Career

Ce module définit les piliers éthiques et les structures de carrière au sein de l'industrie de la cybersécurité.

## ⚖️ Principes Fondamentaux de Sécurité
- **Triade CIA** : Confidentialité, Intégrité, Disponibilité (Availability).
- **Défense en profondeur** : Multiplier les couches de sécurité pour ne pas dépendre d'un seul outil.
- **Le moindre privilège** : Donner uniquement les accès nécessaires à un utilisateur pour son travail.

## 🎓 Parcours et Impact
- **Rôles en Cyber** : Compréhension des différences entre Analyste SOC (Blue Team), Pentester (Red Team) et Ingénieur Sécurité.
- **Éthique** : L'importance du cadre légal et de l'autorisation explicite avant tout test d'intrusion.
- **Formation continue** : L'impact de l'apprentissage constant sur l'efficacité des équipes techniques.

## 📝 Ce que j'ai retenu
La cybersécurité n'est pas qu'une question de technique, c'est aussi une question de rigueur et d'éthique. Comprendre la triade CIA est essentiel pour prioriser les risques dans n'importe quelle organisation.
