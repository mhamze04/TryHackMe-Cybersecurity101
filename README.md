# 🛡️ TryHackMe - Parcours Cybersecurity 101

Bienvenue sur mon dépôt dédié à ma formation en cybersécurité. Ce projet documente ma progression, mes notes techniques et les méthodologies apprises durant le parcours **Cybersecurity 101** de TryHackMe.

## 🎯 Objectif du Projet
L'idée est de centraliser l'ensemble des compétences acquises (Offensif et Défensif) pour servir de base de connaissances personnelle et de portfolio technique.

---

## 🛠️ Outils Maîtrisés
- **Reconnaissance** : `Nmap`, `Gobuster`, `Google Dorks`
- **Analyse Réseau** : `Wireshark`, `Tcpdump`
- **Exploitation** : `Metasploit Framework`, `SQLMap`, `Netcat`
- **Brute-force** : `Hydra`, `John the Ripper`
- **Analyse Défensive** : `CyberChef`, `SIEM Fundamentals`

## 📝 Note Éthique
Ce dépôt est réalisé dans un but purement éducatif. Aucun "flag" ou réponse directe aux exercices de TryHackMe n'est publié ici afin de respecter l'intégrité de la plateforme. Les tests sont effectués uniquement sur des environnements contrôlés et autorisés.
